# NailVital AI - How to Run

## 1. Backend (FastAPI)
1. Navigate to `nailvital-ai-backend/`
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`
5. Run the server: `python -m uvicorn app.main:app --reload`
   - The API will be available at `http://127.0.0.1:8000`

## 2. Frontend (Android)
1. Open `nailvital-ai-android/` in Android Studio (Iguana or later recommended).
2. Wait for Gradle sync to complete.
3. Ensure you have the `nail_model_quantized.tflite` model in `app/src/main/assets/models/`.
   - You can run the `train_model.py` script in the root directory to generate this model from the provided dataset.
4. Build and run on a physical device (Camera features require a real device).

## 3. AI Model Training
1. Ensure you have `tensorflow` installed: `pip install tensorflow pillow`
2. Run `python train_model.py`
3. Copy the output `nail_model_quantized.tflite` to `nailvital-ai-android/app/src/main/assets/models/`

## 4. Key Features
- **Offline AI**: Inference happens entirely on-device via TFLite.
- **Privacy**: No photos are uploaded to the cloud by default.
- **Multilingual**: Supports English, Hindi, Tamil, and Telugu.
- **Pro Features**: Includes PDF report generation and advanced health tracking.
